# 通用教材提示词模板

返回[课程主页](README.md) · 打开[全部章节提示词](COURSE.md)

替换花括号中的内容后复制使用。

```text
请开始讲解神经网络教材的“{章节编号与名称}”。

前置知识：{前置知识}。
本章目标：{学习目标}。
重点公式与概念：{重点}。

请使用中文教材模式，假设我是初学者，但希望真正从公式层面理解。
请按以下顺序讲解：

1. 直觉问题
2. 符号表
3. 使用的数学假设
4. 公式逐步推导
5. 每一步的标量、向量、矩阵或张量形状
6. 至少一个可以手算的数值例子
7. 算法步骤或伪代码
8. 动画或可交互图
9. 常见错误
10. 三道递进练习，暂时不要给完整答案
11. 本节小结

不要跳过等号之间的关键推导。每出现一个新符号，先定义它的含义和形状。
遇到矩阵乘法时，必须明确写出相乘维度为什么合法。
如果章节较长，先列出小节目录，只讲第一个未完成的小节；等我说“继续”再讲下一节。
开始前先用两个问题快速检查我是否掌握前置知识；如果我不会，就先补充最小必要基础。
```

## 复习提示词

```text
请复习“{章节名称}”，不要重新完整讲课。
先向我提出 5 个由浅到深的问题；根据我的回答定位薄弱点。
然后只重讲我不会的公式，并给一个新的手算例子。
最后给一张“符号—形状—作用”对照表和 3 道复习题。
```

## 批改练习提示词

```text
下面是我对“{章节名称}”练习的答案：

{粘贴答案}

请逐题批改。不要只告诉我对错，要指出错误发生在哪一个等号、维度或概念。
先给提示让我自己修改；只有当我再次请求时，才给完整标准答案。
```
